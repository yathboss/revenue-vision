About the Forecasting Model

Beginner-friendly explanation — professional implementation.

What is XGBoost?

XGBoost is a powerful machine learning algorithm that builds many small decision trees and combines them. Each new tree learns to fix the mistakes of the previous trees. This makes XGBoost accurate and stable.

How do we use XGBoost for time-series?

Time-series means data changes over time (sales per week/month). XGBoost needs a normal “table” of features, so we convert time into features.

Features used (simple)

  • Lag features: previous sales values (e.g., last month, 3 months ago).
  • Rolling averages: average sales over recent windows (e.g., last 3 months).
  • Date features: month, quarter, week number, and seasonality (sin/cos).
  • Season flags: Q4 and Nov/Dec indicators.

How multi-step forecasting works

We predict the next period, then add that prediction back into the history, and predict the next one. This is called recursive forecasting.
