How to Use

  1. Select prediction type: Weekly / Monthly / Yearly.
  2. Choose filters: Category, Region, Segment (or All).
  3. Click “Run Forecast”: view chart, KPIs, insights, and download CSV.

When to use Fast vs Advanced

  • Fast mode: instant results using precomputed forecasts + caching.
  • Advanced mode: recompute forecasts for your exact query (cached after first run).
If you select filters with no data, the app will show a friendly message. Try selecting All.
